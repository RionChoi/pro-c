# 세션 12: 정렬 & 탐색 (Sorting & Searching)

## 학습 목표
- 주요 정렬 알고리즘(버블, 선택, 삽입)의 원리를 이해하고 직접 구현한다.
- 이진 탐색과 BST의 개념을 학습한다.
- 시간 복잡도를 비교한다.

## 핵심 개념
- **버블 정렬**: 인접 요소 비교 후 교환, O(n²)
- **선택 정렬**: 최소값을 찾아 앞으로 배치, O(n²)
- **삽입 정렬**: 적절한 위치에 삽입, O(n²) (최선 O(n))
- **시간 복잡도**: 알고리즘 성능을 나타내는 표기법

## 과제 1 (homework.c)
3가지 정렬 알고리즘을 구현하고, 난수 배열로 성능(소요 시간)을 비교한다.

### 실행 예시
```
원본 배열: 42 17 89  ...
버블 정렬: 17 42 89 ...
선택 정렬: 17 42 89 ...
삽입 정렬: 17 42 89 ...
```
