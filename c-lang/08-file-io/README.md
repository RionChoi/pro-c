# 08 파일 I/O (File I/O)

## 과제 1 (`homework.c`)
- 사용자가 파일 이름을 입력받고, 그 파일에 들어있는 정수들을 모두 읽어 합을 구합니다.
- 파일이 존재하지 않으면 `fprintf(stderr, "파일을 열 수 없습니다\n");` 로 오류를 출력하고 종료합니다.
- 모든 입출력은 `fopen`, `fscanf`, `fclose` 로 수행합니다.

## 과제 2 (`homework2.c`)
- 파일에 저장된 정수들을 동적으로 배열에 읽어들인 뒤, 평균을 계산합니다.
- 처음에 `calloc` 로 10개의 정수를 할당하고, 필요할 경우 `realloc` 로 크기를 늘립니다.
- 파일이 비어 있으면 `fprintf(stderr, "빈 파일\n");` 로 오류를 출력합니다.

## 미니 게임 (`game1.c`)
- 사용자가 추측하는 숫자를 파일에 저장하고, 게임이 끝난 뒤 최고 기록(가장 적은 시도)을 파일에 기록합니다.
- `fgets` 로 사용자 입력을 받고, `fprintf` 로 파일에 기록합니다.
- 게임 로직은 1~100 사이의 난수를 생성하고, 사용자가 맞출 때까지 시도 횟수를 셉니다.

### 코드 컨벤션
- `int main(void)` 사용
- `scanf` 반환값 검증
- 문자열 입력에는 `fgets` 사용
- 오류는 `fprintf(stderr, ...)` 로 출력
