# 세션 11: 연결 리스트 (Linked List)

## 학습 목표
- 동적 데이터 구조인 연결 리스트의 개념을 이해한다.
- 노드 생성, 삽입, 삭제, 탐색, 역순 정렬을 구현한다.
- 배열과의 차이점을 체감한다.

## 핵심 개념
- **노드(Node)**: 데이터 + 다음 노드 포인터로 구성된 구조체
- **Head**: 리스트의 첫 번째 노드
- **malloc / free**: 동적 메모리 할당 및 해제
- **NULL 체크**: 빈 리스트, 마지막 노드 판단

## 과제 1 (homework.c)
사용자로부터 정수를 계속 입력받아 연결 리스트에 추가하고,
"0"을 입력하면 전체 리스트를 출력한 뒤 프로그램을 종료한다.

### 실행 예시
```
정수를 입력하세요 (0 입력 시 종료): 10
정수를 입력하세요 (0 입력 시 종료): 20
정수를 입력하세요 (0 입력 시 종료): 5
정수를 입력하세요 (0 입력 시 종료): 0
리스트 내용: 10 -> 20 -> 5 -> NULL
```
