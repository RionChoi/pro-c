# C 세션 10: 비트연산 & 고급

## 학습 목표
- 비트 AND/OR/XOR/SHIFT 연산을 이해한다.
- 비트 마스크로 플래그를 켜고 끄는 방법을 익힌다.
- 비트연산을 활용한 간단한 게임 로직을 구현한다.

## 파일 구성
- `homework.c`: 비트연산 결과 확인
- `homework2.c`: 권한 플래그 비트마스크 실습
- `game1.c`: XOR 기반 숫자 추리 게임

## 실행 예시
```bash
clang -std=c11 -Wall -Wextra -Wpedantic homework2.c -o homework2
./homework2
```
