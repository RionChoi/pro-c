# 강의 4: 포인터와 문자열 기초

## 학습 목표
- 포인터의 의미(주소, 역참조)를 이해한다.
- 배열과 포인터의 관계를 설명할 수 있다.
- 문자열(`char[]`)을 안전하게 입력/출력/순회할 수 있다.

## 핵심 개념
1. **포인터 선언**
   - `int *p;` : 정수형 데이터를 가리키는 포인터
2. **주소 연산자 / 역참조 연산자**
   - `&x` : 변수 `x`의 주소
   - `*p` : 포인터 `p`가 가리키는 값
3. **배열과 포인터 관계**
   - 배열 이름은 첫 원소 주소로 취급됨
   - `arr[i]`는 내부적으로 `*(arr + i)`와 같은 의미
4. **문자열 안전 입력**
   - `scanf("%s", ...)`는 공백 처리/버퍼 초과 이슈가 있으므로 학습 단계에서도 `fgets`를 우선 권장

## 예제 코드
```c
#include <stdio.h>

int main(void) {
    int x = 10;
    int *p = &x;

    printf("x = %d\n", x);
    printf("&x = %p\n", (void *)&x);
    printf("p = %p\n", (void *)p);
    printf("*p = %d\n", *p);

    *p = 20;
    printf("변경 후 x = %d\n", x);

    return 0;
}
```

## 실습
1. 정수 배열 5개를 입력받아 포인터 순회로 합계 계산
2. 이름 문자열을 입력받아 길이를 직접 계산(`strlen` 없이)
