# 강의 2: 조건문과 반복문

## 핵심 개념
- `if / else if / else`
- `for`, `while`
- 누적합, 카운팅 패턴

## 실습
1) 1~100 짝수 합 구하기
2) 입력한 수의 팩토리얼 계산

## 과제 1
- 파일명: `homework.c`
- 요구사항:
  - 메뉴 기반 프로그램
    1) 구구단 출력
    2) 1~N 합
    3) 소수 판별
  - 메뉴는 0 입력 시 종료
  - 잘못된 메뉴 입력 시 에러 메시지 출력

## 과제 2
- 파일명: `homework2.c`
- 요구사항:
  - 높이 N 입력(1~20)
  - 별 피라미드 출력
  - 입력 범위 검증

## 게임 1
- 파일명: `game1.c`
- 내용: 가위/바위/보 (사용자 vs 컴퓨터)
