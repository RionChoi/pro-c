# 강의 1: 변수와 입출력

## 핵심 개념
- `int`, `double`, `const`
- `printf` 포맷: `%d`, `%.1f`
- `scanf` 기본 입력

## 핵심 포인트
- `main`은 `int main(void)` 형태 권장
- 입력값은 항상 검증해야 안전함

## 실습
1) 나이, 키, 이름을 입력받아 출력
2) 정수 2개 입력받아 사칙연산 출력

## 과제
- 파일명: `lesson1_homework.c`
- 요구사항:
  - 사용자에게 정수 2개 입력
  - 덧셈/뺄셈/곱셈/나눗셈(몫, 나머지) 출력
  - `scanf` 반환값 검사 실패 시 `입력 오류` 출력 후 종료
