# Session 1: Introduction to PyTorch Tensors

This session introduces the fundamentals of PyTorch, focusing on Tensor creation, manipulation, and basic operations.

## Objectives
* Understand what a PyTorch Tensor is.
* Learn how to create tensors from data (lists, NumPy arrays).
* Practice basic tensor operations (addition, multiplication, indexing).

## Setup
Ensure you have PyTorch installed.
```bash
pip install torch
```
