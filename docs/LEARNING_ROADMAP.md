# pro-c 학습 로드맵 (C → C++ 통합 과정)

## 목표
- C 문법 기초부터 고급까지 체계적으로 습득한다.
- C 완료 후 C++를 이어서 학습한다.
- 매일 아침 7시(KST) 자동으로 2세션씩 코딩하고 GitHub에 푸시한다.
