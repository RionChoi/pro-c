# pro-c 학습 로드맵 (C → C++ 통합 과정)

## 목표
- C 문법 기초부터 고급까지 체계적으로 습득한다.
- C 완료 후 C++를 이어서 학습한다.
- C 20세션과 C++ 50세션으로 구성된 핵심 학습 과정을 완료 상태로 관리한다.
- Phase 6 확장 과정은 C++26과 프로덕션 C++ 주제로 별도 추적한다.
