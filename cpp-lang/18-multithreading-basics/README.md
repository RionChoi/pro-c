# 🚀 C++-18: 멀티스레딩 기초 (Multithreading Basics)

## 📖 개요
멀티스레딩은 여러 작업을 동시에 실행하여 프로그램의 처리 효율성과 반응성을 극대화하는 기술입니다. 이 세션에서는 C++11부터 도입된 `<thread>` 라이브러리를 사용하여 기본적인 스레드 생성, 관리 및 동기화 방법을 학습합니다.

## 💡 주요 개념
1.  **`std::thread`**: 새로운 실행 스레드를 생성하고 관리하는 기본 클래스입니다.
2.  **스레드 시작**: 함수 또는 callable 객체를 인수로 전달하여 스레드를 시작합니다.
3.  **동기화 (Synchronization)**: 다중 스레드가 공유 자원에 동시에 접근할 때 발생하는 **경쟁 조건(Race Condition)**을 막기 위해 반드시 동기화 메커니즘이 필요합니다.
    *   **`std::mutex`**: 뮤텍스를 사용하여 특정 코드 블록(Critical Section)에 한 번에 하나의 스레드만 접근하도록 보장합니다.
    *   **`std::lock_guard` / `std::unique_lock`**: 복잡한 수동 `lock()`/`unlock()` 호출 대신, RAII 패턴을 사용하여 스코프를 벗어날 때 자동으로 락을 해제(Unlock)해줍니다. 이는 스레드 관련 버그를 예방하는 가장 안전한 방법입니다.

## 📝 과제 목표
간단한 카운터나 데이터 처리를 여러 스레드가 동시에 수행하도록 구현하고, 뮤텍스(`std::mutex`)를 사용하여 **경쟁 조건 없이** 데이터의 무결성을 유지하는 연습을 합니다.
