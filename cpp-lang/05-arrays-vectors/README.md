# Session 05: 배열 & 벡터 (Arrays & Vectors)

## 학습 목표
- C++에서 정적 배열과 `std::vector`의 차이를 이해한다.
- `std::vector`를 활용한 동적 배열 관리법을 익힌다.
- 범위 기반 for문과 인덱스 기반 순회를 모두 다룬다.

## 핵심 개념

### 정적 배열 vs std::vector
```cpp
// C 스타일 정적 배열
int arr[5] = {1, 2, 3, 4, 5};

// C++ std::vector (동적 크기 조절 가능)
std::vector<int> vec = {1, 2, 3, 4, 5};
vec.push_back(6);           //末尾に要素を追加
vec.insert(vec.begin(), 0); // 先頭に挿入
```

### 주요 멤버 함수
| 함수 | 설명 |
|------|------|
| `push_back()` |末尾に要素を追加 |
| `pop_back()` |末尾の要素を削除 |
| `size()` |要素数を返す |
| `empty()` |空인지 확인 |
| `clear()` |全要素を削除 |
| `erase()` |特定位置の要素を削除 |

### 범위 기반 for문
```cpp
for (int val : vec) {
    std::cout << val << " ";
}
```
