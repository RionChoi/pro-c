# 세션 03: 제어 흐름 (Control Flow)

## 학습 내용
- if, else if, else 조건문
- switch-case 문
- for, while, do-while 반복문
- break, continue, return 제어
- 삼항 연산자 (?:)
- 중첩 루프와 레이블

## 핵심 개념
C++의 제어 흐름은 C 언어와 동일하지만, bool 타입과 std::cin을 활용한 입력 처리有所不同.

## 과제 1 (homework.cpp)
사용자로부터 점수(0~100)를 입력받아 학점을 출력하는 프로그램을 작성하세요.
- 90점 이상: A
- 80점 이상: B
- 70점 이상: C
- 60점 이상: D
- 60점 미만: F

## 과제 2 (homework2.cpp)
구구단을 출력하되, 사용자가 선택한 단만 출력하는 프로그램을 작성하세요.
- 1~9 사이의 숫자가 아니면 재입력 요청

## 미니 게임 (game1.cpp)
숫자 맞추기 게임을 구현하세요.
- 컴퓨터가 1~100 사이의 숫자를 무작위로 선택
- 사용자가 숫자를 추측하여 맞추는 게임
- 횟수 제한 7회, 남은 횟수 표시
