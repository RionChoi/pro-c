# 🚀 C++-17: 스마트 포인터와 이동 시맨틱 (Smart Pointers & Move)

## 📖 개요
이 세션은 C++의 메모리 관리와 성능 최적화에 필수적인 **스마트 포인터**(`std::unique_ptr`, `std::shared_ptr`)와 **이동 시맨틱**을 학습합니다.

## 💡 주요 개념
1.  **스마트 포인터**: RAII(Resource Acquisition Is Initialization) 원칙을 적용하여 동적 메모리 누수(Memory Leak)를 방지합니다.
    *   `std::unique_ptr`: 단독 소유권을 가집니다. 객체가 유효 범위(scope)를 벗어나면 자동으로 메모리를 해제합니다.
    *   `std::shared_ptr`: 공유 소유권을 가집니다. 참조 카운트(Reference Count)를 사용하여, 참조하는 포인터가 0개가 되면 자동으로 메모리를 해제합니다.
2.  **이동 시맨틱 (Move Semantics)**: 효율적인 자원 이전(Resource Transfer)을 가능하게 합니다.
    *   C++11에서 도입되었으며, 임시 객체나 이동 가능한 객체에서 자원(Resource)을 '훔쳐' 오는 방식으로 복사(Copy) 대신 **이동(Move)**을 발생시켜 불필요한 대용량 데이터의 복사 비용을 제거합니다.
    *   rvalue reference (`&&`)와 이동 생성자/이동 대입 연산자(`std::move`)가 핵심입니다.

## 📝 과제 목표
주어진 클래스에서 수동 `new`/`delete` 대신 스마트 포인터를 사용하여 메모리 관리를 수행하고, `std::move`를 적절히 사용하여 불필요한 복사를 최소화하는 연습을 합니다.
